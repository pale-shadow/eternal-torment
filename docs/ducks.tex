\section{dm-with-ducks}
This section documents explorations inspired by \textit{Discrete Mathematics with Ducks}, focusing on the intersection of formal proof-writing and algorithmic implementation. Projects here bridge the gap between abstract mathematical themes and concrete code execution on the NVIDIA Jetson cluster.

\subsection*{Current Focus Areas}
\begin{itemize}
    \item \textbf{Graph Traversals:} Implementing Dijkstra's and spanning tree algorithms to optimize data flow across cluster nodes.
    \item \textbf{Counting \& Complexity:} Analyzing the efficiency of cryptographic implementations (e.g., Vigenere and Rail Fence) through a combinatorial lens.
    \item \textbf{Number Theory:} Exploring RSA encryption and decryption logic as a natural progression from the classical ciphers already documented.
\end{itemize}
